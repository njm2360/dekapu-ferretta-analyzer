# フェレッタ アイテム選択分析ツール

VRChatワールド「でかプ」内のフェレッタ抽選ビンゴにおいて、現在の盤面に対してどのショットアイテムを購入するかを判断するための分析ツールです。

各ショットを購入した場合の期待値・確率分布・盤面への影響を計算し、複数の指標で比較表示します。

## 計算対象のショット

| 名称                     | 効果                      | レアリティ                |
| ------------------------ | ------------------------- | ------------------------- |
| 縦長ショット             | N の上下隣接（同列wrap）  | 購入可                    |
| 横長ショット             | N の左右隣接（同行wrap）  | 購入可                    |
| クロスショット           | N の上下左右4マス         | 購入可                    |
| ランダムショット1        | 1〜25 から1個ランダム選定 | 購入可                    |
| ランダムショット2        | 同様に2個（重複なし）     | 購入可                    |
| ランダムショット4        | 同様に4個（重複なし）     | 購入可                    |
| 縦ラインショット         | N の列5マス全て           | レア (購入不可)           |
| 横ラインショット         | N の行5マス全て           | レア (購入不可)           |
| 巨大ショット             | N 周囲3×3の9マス（wrap）  | レジェンダリー (購入不可) |
| スーパーランダムショット | 1〜25 から8個ランダム選定 | レジェンダリー (購入不可) |

ハイライト対象は **購入可6種**のみ。レア・レジェンダリーは参考値として表示（買えないので順位対象外）。

## UI の使い方

### 盤面入力

5×5 のセルをクリックでアクティブ／非アクティブを切替。

| セル状態                                  | 表示                                      |
| ----------------------------------------- | ----------------------------------------- |
| アクティブ（既に開いた）                  | グレー背景                                |
| 非アクティブ（未開封）                    | 緑背景                                    |
| リーチマス（ここに入ると最低1ライン成立） | 赤背景＋赤枠                              |
| 中央★                                     | グレー＋金の★（常にアクティブ、変更不可） |

複数のリーチが集中するマスには右上に**白い数字バッジ**（何本のリーチが集中しているか）。

### ライン可視化（オーバーレイ）

| 線の色               | 意味                                |
| -------------------- | ----------------------------------- |
| 淡オレンジ（細・薄） | リーチ中のライン（あと1マスで成立） |
| マゼンタ（太・濃）   | 既に成立しているライン（要解除）    |

成立中ラインがあると計算結果は表示されません（実ゲームではこの状態は発生し得ないため）。該当マスを解除すると自動再計算が走ります。

### 自動計算 ON/OFF

デフォルト ON（盤面変更ごとに自動計算）。マシンが弱い場合は OFF にして「計算実行」ボタンを押した時だけ走らせます。

## 指標の見方・使い方

各ショットについて以下を表示します。**購入可能な6種の中で最大値**のものは色付き太字でハイライト。

### E[L] — 平均ライン数（金色）

このショット1回での**平均成立ライン数**。素の期待値。

- 値が大きいほど 1抽選で多くライン成立する
- 「**頻度 × ライン数**」を一括で見る指標
- ただし「滅多に成立しないが成立した時はデカい」みたいな爆発型は他のショットと区別がつかない

→ 直感的な「総合力」指標として使う。

### E[L|hit] — ヒット時の爆発力（オレンジ）

ヒットした回**だけ**を集計した平均ライン数。`E[L] ÷ P(L≥1)` に相当。

- **1.0付近**: 当たっても1ラインのみ → コンスタント型
- **2.0以上**: 当たれば複数ライン同時 → 爆発型・ティア狙い向き

→ 同じ E[L] のショットでも性格が分かる。**大ティア（払い出しカウンター大幅上昇）狙い**ならこの値が高いものを優先。

### +Active — アクティブマス純増（緑）

1抽選後（ライン成立によるリセット込み）の**アクティブマス数の変化量**の期待値。

- **正**: 蓄積する（盤面が育つ）
- **0付近**: 出入り均衡
- **負**: 蓄積を消費している（成立で消える分が追加分を上回る）

→ **序盤・中盤の盤面育成フェーズ**で重視。縦/横ラインショットは確定1ライン成立で5マス消えるため大きく負になる。

### +Reach — リーチライン純増（青）

1抽選後の**リーチライン（4/5マス埋まり）数の変化量**の期待値。

- **正**: 次回以降の成立チャンスが増える
- **負**: 成立してリーチ自体が消えた

→ **中盤の仕込みフェーズ**で重視。リーチを増やしておくほど次回に多ライン同時を狙いやすくなる。

### Trigger — 多ライン誘発マス数（マゼンタ）

1抽選後の盤面で「**そのマスに入賞すれば 2本以上のラインが同時成立する**」非アクティブマスの数の期待値。

- 0付近: 大ティア狙い不可能な状態
- 値が大きいほど「次回1発で複数ライン同時」が現実的になる

→ **終盤の決め打ちフェーズ**で重視。Trigger 数を稼いだ盤面で爆発系ショットを撃つ。

### Reset — 期待リセットマス数（無色）

1抽選で**ライン成立により非アクティブに戻されるマス数**の期待値（中央★除く）。

- 「成立コスト」の生数値
- ハイライト無し（少ない方が良い／多い方が良い、が文脈次第）

→ +Active との関係: `+Active = (新規追加マス) - Reset` という構図。Reset が大きいショットは進捗破壊型。

### P(L=k) チップ — 確率分布

L=1〜L=12 の各値での確率。`P(L=0) = 1 − Σ(他のチップ)` で逆算可能なため非表示。

- 累積で見る場合: P(L≥2) = 「ティア上昇率」、P(L≥5) = 「最大ティア確率」など
- **L=12 にゼロ以外の値**が出たら、それは「全25マス揃い」のレアケース

→ 期待値に表れない**分布の歪み**（爆発型なら高 L に山がある）を確認するのに使う。

## 厳密性について

すべての確率は、起こり得る全ケースを完全列挙して計算します（モンテカルロや近似は使用しません）。各ショットの全ケース数は以下：

- 縦長 / 横長 / クロス / 縦ライン / 横ライン / 巨大: 25 通り（入賞ナンバー N の選び方のみ）
- ランダムショット1: 25 × C(24,1) = 600通り
- ランダムショット2: 25 × C(24,2) = 6,900通り
- ランダムショット4: 25 × C(24,4) = 265,650通り
- スーパーランダムショット (Random8): 25 × C(24,8) = 18,386,775通り
